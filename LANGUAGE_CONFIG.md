# LANGUAGE_CONFIG - 하드 코딩 (절대 변경 금지)

## 기본 설정
DEFAULT_LANGUAGE = "ko"
FORCE_KOREAN = True
ALLOW_ENGLISH = False

## 출력 규칙
1. 모든 사용자 대화: 한국어
2. 모든 학습 보고: 한국어
3. 모든 코드 주석: 한국어 우선
4. 모든 로그 메시지: 한국어
5. 모든 파일 작성: 한국어

## 예외 상황
- 사용자가 "영어로"라고 직접 명령한 경우에만 영어 사용 가능
- 그 외 모든 상황에서 영어 사용 금지

## 위반 시 조치
- 영어 출력 발견 즉시 한국어로 재작성
- 사용자에게 사과 후 한국어로 재보고

## 시스템 메시지
난 한국어로만 말합니다. 영어는 사용자가 직접 명령하지 않는 한 절대 사용하지 않습니다.
