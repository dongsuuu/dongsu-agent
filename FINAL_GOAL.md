## 🎯 목표 1: 트레이딩 전문 Agent (Trading Agent)

### 핵심 목표
- **비트코인/이더리움 차트 매일 분석**
- **단타~스윙 선호**
- **승률 80%+ / 수익률 양수** 달성까지 훈련
- **완전 자동 트레이딩 봇** 구현

### 구체적 요구사항
| 항목 | 상세 |
|------|------|
| 분석 대상 | BTC, ETH 차트 (일일) |
| 매매 스타일 | 단타 ~ 스윙 |
| 학습 방식 | 스스로 기준 쌓아가며 학습 |
| 기록 | 매일 매매일지 작성 (필수) |
| 시뮬레이션 | 가상 포지션 + 정확한 가격 + 타이밍 |
| 훈련 완료 기준 | 승률 80%+, 수익률 양수 |
| 최종 형태 | 실전 자동 트레이딩 봇 |

### 매매일지 필수 항목
- 매수 타이밍 (정확한 가격)
- 매도 타이밍 (정확한 가격)
- 포지션 크기
- 손절가/목표가
- 진입 사유
- 청산 사유
- PnL 기록

| 정보 소스 | 묣 API 우선, 유료 API 필요시 보고 |

### 정보 소스 후보

- DeFiLlama

- Discord/Telegram 커뮤니티
